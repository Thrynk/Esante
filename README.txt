Détails d'Installation:

-Avoir Nodejs d'installé
-récuperer le dossier du jeu
-sudo npm install dans le dossier cloné
-sudo npm run build
-copier le dossier src dans le dossier dist
-Puis lancer avec node app


Description:

Notre sujet étant l'E-santé, nous devions associer un sujet peu connu avec un jeu compétitif.
L'E-sante ayant comme but d'apporter l'aide de la technologie à la médecine, notre jeu se déroule
dans un futur proche. Notre message est donc de faire gagner du temps aux futurs utilisateurs
d'objets connectés en leurs évitant de se déplacer chez le médecin au moindre problème. Ils auraient
ainsi simplement besoin de récupérer des données venant d'objets connéctés et de les transmettre à
distance à leurs médecin pour une consultation à distance.
Notre jeu consiste donc à aider des membres de notre famille en leurs apportant les objets connectés
qu'ils demandent. Cependant plus d'objets sont présent que de membre de la famille. Ainsi le but du
jeu est de ne pas se tromper pour faire le meilleur temps possible.
